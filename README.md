# ComeCome

## 各種CRUD処理
- ログイン認証
- 新規登録
- メールアドレス認証
- 住所自動入力
- 画像アップロード、プレビュー
- 予約機能
- 現在地、及び登録店舗の住所をピンドロップ表示
- ゲストログイン機能
- ajax
